Pattern Design Rationale

Pattern Visual Why This Works
Open Empty white Obvious absence of terrain
Blocking Solid dark Impenetrable = solid mass
Cover Sparse dots Things to hide behind, scattered obstacles
Difficult Diagonal stripes Classic "caution" pattern, impediment
Dangerous X marks Universal hazard symbol
Cover + Difficult Dense stipple More dots = more stuff (cover + impediment)
Cover + Dangerous Rings/circles Enclosed danger zone, distinct from X
Difficult + Dangerous Chevrons Angular = aggressive, warning arrows
All Three Crosshatch grid Busy = "everything applies here"
